# 合成大西瓜（网页版）

## 如何打开

**推荐**：在项目文件夹里启动本地服务器后，用浏览器访问。

```bash
cd "/Users/wangruoxuan/Downloads/合成大旅店"
python3 -m http.server 8080
```

浏览器打开：http://localhost:8080

也可以直接双击 `index.html` 打开（已内置 `matter.min.js`，不依赖外网）。

## 内置资源（assets 目录）

- `小球图片/`：11 级小球图（游戏默认）
- `背景图片/`：放入图片后运行 `python3 scripts/update-manifest.py`，每局随机一张铺满背景
- `背景音乐/`：页面 BGM「不万能的喜剧」（已转 m4a，约 20% 音量循环；打开页面自动播放，必要时点一下游戏区）
- `音效/`：合成时随机播放（约 40% 音量）
- `店学小知识.docx`：复活题来源（已写入 `shop-quiz.js`）

## 操作

- 鼠标/手指左右移动定位
- 点击或松手投放
- 相同水果碰撞会合成更大一级
- 水果稳定后超过顶部红线约 2 秒则游戏结束
